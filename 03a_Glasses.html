Usage

This project is mainly dedicated to the people who do not have a good eyeseight as we do.

That is why I have created these pair of glasses for the visually challenged.

Working

When the Ultrasonic sensor senses the object by the reflection of Ultrasonic waves, it then sends the signal to Arduino Nano which inturn sends it to the buzzer and vibration motor. As a result the buzzer starts beeping and the vibration motor starts vibrating. Thus, alerting the person to switch directions.

Rules for the code

  1. Copy and paste this code in the Arduino coding software.
  2. Connect the cable of the Arduino to the Adruino.
  3. Check the code by clicking on the Verify button(✅).
  4. Execute the code by clickin on the upload button(➡️).

If you want to function the project by using a battery first execute the code then connect a 9 volt battery to the other cable using a battery cap.
